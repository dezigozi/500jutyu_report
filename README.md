# 架装品 500受注レポート

500受注実績 Excel を読み込み、部店・顧客・担当者・品番までドリルダウンして売上・粗利を確認するデスクトップアプリです。

## ドキュメント

**→ 起動方法・画面機能・列対応・トラブルシューティングは [`マニュアル.md`](./マニュアル.md) にまとめています。**

## すぐに開発起動する

```bash
cd lease-report-app
npm install
npm run dev
```

または `dev.bat` をダブルクリック。アイコン付きショートカットは `ショートカット作成.bat` または `npm run shortcut:dev`（詳細はマニュアル参照）。
